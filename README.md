El objetivo de esta iniciativa es crear una estructura similar a la que tiene la LPI (lega pauper italia) aquí en Uruguay (y luego tal vez la región).

Otro objetivo es consolidar a todos los jugadores de pauper del Uruguay y generar sinergia entre ellos.

Como ingeniero de software me gustaría concentrarme primero en los aspectos técnicos para desarrollar el software necesario que le de soporte a esta iniciativa.
